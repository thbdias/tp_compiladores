*Enumeracao das classes dos tokens da linguagem L


1  - atribuição (=) -> (Atrib)
2  - tipos de dados (string boolean byte integer) -> (TDados)
3  - comando if (if)
4  - comando else (else)
5  - comando repetição (while) 
6  - comando leitura (readln )
7  - comando escrita (write	writeln)
8  - abre parenteses ( '(' ) -> (AParent)
9  - fecha parenteses ( ')' ) -> (FParent)
A - operador relacional ( > < <> >= <= ==) -> (ORel)
B - separador (,) -> (Sep)
C - operador aritmético (+ - * /) -> (OArit)
D - abre chaves ({) -> (ACha)
E - fecha chaves (}) -> (FCha)
F - expressão (valores dos tipos "string" ou true-false inteiros ou byte) -> (Exp)
G - const (palavra 'const') -> (Const)
H - identificadores -> (ID)
I - id/byte -> (ID-Byte)
J - operador lógico (and or) -> (OLog)
K - delimitador de comando (;) -> (DComand)
