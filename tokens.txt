*Enumeracao das classes dos tokens da linguagem L


1  - atribuição (=)
2  - tipos de dados (string boolean byte integer)
3  - comando if (if)
4  - comando else (else)
5  - comando repetição (while) 
6  - comando leitura (readln )
7  - comando escrita (write	writeln)
8  - abre parenteses ( '(' )
9  - fecha parenteses ( ')' )
A - operador relacional ( > < <> >= <= ==)
B - separador (,)
C - operador aritmético (+ - * /)
D - abre chaves ({)
E - fecha chaves (})
F - expressão ( decimal )
G - const (palavra 'const')
H - identificadores
I - id/hexadecimal
J - operador lógico (and or not)
K - delimitador comando (;)
L - expressão ( booleana )
M - expressão ( hexadecimal )
N - expressão ( string )
P - comando then (then)
